# Novel Translator - 双语小说翻译系统

## 项目简介

基于 Java Spring Boot + Python Agentscope 的多语言小说翻译系统，支持三种翻译模式：快速模式、专家模式、团队模式。

## 快速开始

### 环境要求

- Java 21（启用预览特性）
- Python 3.10+
- MySQL 8.0
- Redis 7.0+

### 启动服务

```bash
# 1. 启动 Spring Boot 后端（端口 8080）
mvn spring-boot:run

# 2. 启动 Python 翻译微服务（端口 8000）
cd services/translate-engine
python translate_server.py

# 3. 启动前端（如已配置）
cd web-app
npm install && npm run dev
```

## 翻译模式对比

| 特性 | 快速模式 (fast) | 专家模式 (expert) | 团队模式 (team) |
|------|----------------|------------------|----------------|
| **管线** | 缓存 + 直译 | 完整四级管线 | AI 多 Agent 协作 |
| **缓存** | 仅 L1 | 模式层级缓存 | team 专属缓存 |
| **RAG** | 跳过 | L2 语义匹配 | L2 语义匹配 |
| **实体一致性** | 跳过 | L3（分段提取） | L3（分段提取） |
| **翻译引擎** | 直译 | 直译 | MsgHub 群聊协作 |
| **状态流转** | 直接完成 | 直接完成 | SUBMITTED（待审校） |
| **适用场景** | 网页翻译 | 文档翻译 | 小说协作项目 |

### 缓存命中优先级

```
快速模式: team → expert → fast（高质量优先）
专家模式: team → expert
团队模式: team（仅团队缓存）
```

## AI 翻译团队（团队模式）

### 架构

```
章节翻译流程:
  1. L1/L2/L3 三级缓存检查 → 命中则返回
  2. RAG 语义匹配 → directHit >= 0.85 则返回
  3. 分段提取实体 → 翻译实体 → 构建占位符 [{1}], [{2}]...
  4. AI 翻译团队 (Python Agentscope MsgHub):
     ├─ Translator（主翻译）
     ├─ Terminologist（术语审校）
     └─ Polisher（语言润色）
  5. 占位符还原 → 写入缓存 → 存储 RAG 记忆
  6. 章节状态 → SUBMITTED（待审校）
```

### 分段策略

| 条件 | 策略 |
|------|------|
| 文本 ≤ 5000 字 | 不分段，一次性处理 |
| 文本 > 5000 字 | 按 3000 字/段，段落边界优先，句子边界兜底 |
| 团队模式 | 按章节隔离，章节超长时内部分段 |

### 提示词模板

- **战斗热血** (`battle`): 战斗、奇幻、玄幻类
- **推理悬疑** (`mystery`): 推理、悬疑、惊悚类
- **日常/其他** (`daily`): 恋爱、校园、生活类

## 团队协作工作流

```
[AI翻译中] → [待审校] → [审校中] → [待终审] → [已发布]
                ↑___________↓ (打回修改)
```

### 角色权限

| 角色 | 权限 | 动作 |
|------|------|------|
| **项目Owner** | 创建项目、上传文件、分配章节、导出 | 发起翻译任务，配置AI策略 |
| **审校员** | 查看AI译文、修改、批注、标记问题 | 修AI的错误 |
| **终审** | 确认发布、锁定章节 | 质量背书 |

### 断点续传

- 服务重启后自动恢复 TRANSLATING 状态的章节
- 单章最大重试 3 次，超次标记为 REJECTED
- 实体映射表和 AI 术语表持久化到数据库

## 数据库表

| 表名 | 用途 |
|------|------|
| `collab_project` | 协作项目 |
| `collab_chapter_task` | 章节任务 |
| `collab_project_member` | 项目成员 |
| `collab_comment` | 批注评论 |
| `translation_cache` | 翻译缓存（含 mode 字段） |
| `chapter_entity_map` | 章节实体映射（持久化） |
| `ai_glossary` | AI 维护的术语表 |

## 术语优先级

```
用户术语表 > AI 术语表（confirmed） > AI 自动提取 > LLM 翻译
```
