## Overview（个人简介正文）

I'm a full-stack developer who builds production-ready SaaS platforms from scratch — from architecture design and payment integration to deployment and monitoring.

### What I Deliver

**End-to-end SaaS development**
I don't just write code — I own the full stack. A recent example: I built a multi-tenant translation platform with a React web app, Chrome browser extension, Stripe subscription billing (Checkout + Webhooks + Billing Portal), team collaboration workspace, and a public REST API with API-key access. The entire system runs on Docker Compose with Nginx as the gateway.

**Secure payment integration**
Stripe Checkout with webhook-driven subscription state sync, idempotent event handling, signature verification, and graceful fallback when webhooks fail. I've handled the edge cases: double-charges, missing webhooks, session verification, and cancel-at-period-end logic.

**Browser extension development**
Chrome Extension (Manifest V3) with content scripts, messaging API between popup/service-worker/content pages, DOM-aware full-page translation, reader mode extraction, and inline selection translation — all communicating with a remote backend.

**REST API design**
15+ controllers with JWT + API key dual authentication, tenant context interceptors, SSE streaming for long-running operations, and proper error codes. I also write consumer-facing API documentation with request/response examples.

### How I Work

- **Communicate early**: I share architecture decisions and trade-offs before writing code, so there are no surprises
- **Ship in increments**: Working features over perfect architecture. I deliver testable functionality at every milestone
- **Test what matters**: Unit tests for business logic, integration tests for critical paths (payments, auth), and manual testing for UX
- **Document everything**: API docs, setup instructions, and deployment guides — so the next developer can pick it up without asking me

### I'm a Good Fit When You Need
