支持哪些文件格式？
目前支持 TXT、EPUB、DOCX 和 PDF 格式，单个文件最大 50MB。TXT 格式处理速度最快，EPUB 格式能保留原始排版。
三种翻译模式有什么区别？

快速模式：使用轻量模型，速度最快，适合日常翻译。

专家模式：使用大模型精翻，质量最高，但处理时间较长。

团队模式：多引擎协作 + RAG 记忆库交叉验证，适合对术语一致性要求高的长篇翻译。

我的翻译内容安全吗？
非常安全。所有翻译内容通过加密传输，不会存储到第三方服务器。您的原文和译文仅您本人可见，我们不会分享或泄露任何翻译内容。
如何导出翻译结果？
翻译完成后，在翻译页面点击「下载」按钮即可将译文下载为 TXT 文件。同时您也可以直接复制译文到剪贴板。
如何使用 API 密钥？
在用户中心页面可以找到您的 API 密钥。将密钥添加到请求头中即可调用翻译 API：Authorization: Bearer YOUR_API_KEY
翻译失败了怎么办？
请先检查网络连接是否正常，然后确认文件格式是否在支持范围内。如果仍然失败，可以在翻译历史中找到该文档，点击「重新翻译」按钮重试。如果问题持续，请通过帮助中心联系我们。
